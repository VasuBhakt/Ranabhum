# Submission and Sandboxing Engine

A low-latency tracking system monitoring the interactions between the bots and the contestant's exchange. It accurately measures:

- Latency: Order acknowledgment time (p50, p90, and p99 latencies).

- Throughput: Maximum transactions per second (TPS)
  handled before failure.

- Correctness: Validation of price-time priority and fill accuracy.
