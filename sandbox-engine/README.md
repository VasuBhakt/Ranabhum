# Submission and Sandboxing Engine

A secure pipeline where contestants upload their binaries or source code (e.g., C++, Rust, Go). The platform containerizes and deploys these submissions in strictly isolated environments to prevent malicious code execution and ensure fair resource allocation (e.g., CPU pinning, strict memory
limits).
