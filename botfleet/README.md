# Distributed Bot Fleet

A scalable traffic generation service capable of spawning thousands of distributed bots. These bots will simulate market participants, sending high-velocity FIX, REST or WebSocket requests (Limit Orders, Market Orders, Cancels) to the contestant's endpoint.
